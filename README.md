# LinkVLC-App
LinkVLC, modern, gizlilik odaklı bir medya dinleyici ve oynatıcıdır. HLS/DASH akışlarını izole bir ortamda yakalar ve sinematik bir arayüzde oynatır.

LinkVLC is a modern, privacy-focused media sniffer and player. It captures HLS/DASH streams in an isolated environment and plays them in a cinematic interface.
